# Artifacts

This directory contains public-safe evidence summaries for milestones where
shareable artifacts exist or are planned.

Artifact folders contain descriptions and readiness assessments — not raw
internal dumps. Each folder follows these conventions:

- What artifact exists (or is planned)
- What it proves
- What it does not prove
- Publication readiness
- Where a public-safe version should be hosted

See [docs/artifact-guidelines.md](../docs/artifact-guidelines.md) for
full guidelines.
